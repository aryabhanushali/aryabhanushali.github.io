Projects

Patient Intake Tracker

Improving hospital intake flow with Python and GUI design.

One of my earliest projects, this system streamlines patient check-in and COVID-19 screening using a Tkinter-based interface. It lets patients confirm appointments or register as new patients, collects symptom data, and flags COVID risks for early testing. This project sparked my interest in user-centered healthcare tech.

Tech stack: Python, Tkinter, file I/O, dictionaries, time module
